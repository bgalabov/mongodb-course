# MongoDB Operators Cheat Sheet

## 1. Comparison Operators

### 1. `$eq`
- **Описание:** Съвпада с документи, където стойността на полето е равна на определена стойност.
- **Пример:** `{ "age": { "$eq": 25 } }`  
  Намери документи, където `age` е равно на 25.

### 2. `$ne`
- **Описание:** Съвпада с документи, където стойността на полето не е равна на определена стойност.
- **Пример:** `{ "age": { "$ne": 25 } }`  
  Намери документи, където `age` не е равно на 25.

### 3. `$gt`
- **Описание:** Съвпада с документи, където стойността на полето е по-голяма от определена стойност.
- **Пример:** `{ "age": { "$gt": 25 } }`  
  Намери документи, където `age` е по-голямо от 25.

### 4. `$gte`
- **Описание:** По-голямо или равно.
- **Пример:** `{ "age": { "$gte": 25 } }`

### 5. `$lt`
- **Описание:** По-малко от.
- **Пример:** `{ "age": { "$lt": 18 } }`

### 6. `$lte`
- **Описание:** По-малко или равно.
- **Пример:** `{ "age": { "$lte": 65 } }`

---

## 2. Logical Operators

### 7. `$and`
- **Описание:** Всички условия трябва да са изпълнени.
- **Пример:**  `{ "$and": [ { "age": { "$gt": 18 } }, { "status": "active" } ] }`

### 8. `$or`
- **Описание:** Съвпада с документи, които отговарят на поне едно от условията,
зададени в масива.
- **Пример:** `{ "$or": [ { "age": { "$lt": 25 } }, { "name": "Alice" } ] }` - Намери документи,
където `age` е по-малко от 25 или `name` е "Alice".

### 9. `$not`
- **Описание:** Обратното на условието, зададено вътре в него.
- **Пример:** `{ "age": { "$not": { "$gt": 25 } } }` - Намери документи, където `age` не е поголямо от 25.

### 10. `$nor`
- **Описание:** Съвпада с документи, които не отговарят на нито едно от условията,
зададени в масива.
- **Пример:** `{ "$nor": [ { "age": { "$gt": 25 } }, { "name": "Alice" } ] }` - Намери документи, където `age` не е по-голямо от 25 и `name` не е "Alice".

---

## 3. Elementary operators

### 11. `$exists`
- **Описание:** Съвпада с документи, където полето съществува (или не съществува,
ако е зададено с `false`).
- **Пример:** `{ "middle_name": { "$exists": true } }` - Намери документи, където полето
`middle_name` съществува.

### 12. `$type`
- **Описание:** Съвпада с документи, където полето е от определен тип.
- **Пример:** `{ "age": { "$type": "int" } }` - Намери документи, където полето `age` е от
тип `int`.

---

## 4. Array Operators

### 13. `$all`
- **Описание:** Съвпада с документи, където масивът съдържа всички елементи,
зададени в масива.
- **Пример:** `{ "tags": { "$all": ["red", "blue"] } }` - Намери документи, където масивът
`tags` съдържа и "red", и "blue".

### 14. `$elemMatch`
- **Описание:** Съвпада с документи, където поне един елемент от масива отговаря на
зададените критерии.
- **Пример:** `{ "grades": { "$elemMatch": { "score": { "$gt": 80 } } } }` - Намери документи, където поне един елемент от масива `grades` има `score` по-голямо от 80.

###15. `$size`
- **Описание:** Съвпада с документи, където масивът има точно определен размер.
- **Пример:** `{ "tags": { "$size": 3 } }` - Намери документи, където масивът `tags` има
точно 3 елемента.

---

## 5. Regex operators

### 16. `$regex`
- **Описание:** Съвпада с документи, където стойността на полето съвпада с
регулярния израз.
- **Пример:** `{ "name": { "$regex": "^A" } }` - Намери документи, където `name` започва
с "A".

## 6. Operators, used in Update operations

### 17. `$inc`
 - **Описание:** Увеличава стойността на полето с определена стойност.
 - **Пример:** `{ "$inc": { "age": 1 } }` - Увеличи стойността на `age` с 1.
 
### 18. `$set`
- **Описание:** Задава стойността на полето на определена стойност. Ако полето не съществува, то се добавя.
- **Пример:** `{ "$set": { "name": "Alice" } }` - Задай стойността на `name` на "Alice".

### 19. `$unset`
- **Описание:** Премахва полето от документа.
- **Пример:** `{ "$unset": { "age": "" } }` - Премахни полето `age`.

### 20. `$push`
- **Описание:** Добавя стойност към масив.
- **Пример:** `{ "$push": { "tags": "new_tag" } }` - Добави "new_tag" към масива `tags`.

### 21. `$pull`
- **Описание:** Премахва всички стойности от масив, които отговарят на определено
условие.
- **Пример:** `{ "$pull": { "tags": "old_tag" } }` - Премахни всички елементи "old_tag" от
масива `tags`.
