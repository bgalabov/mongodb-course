# MongoDB Operators Cheat Sheet

## 1. Comparison Operators

### 1. `$eq`
- **Описание:** Съвпада с документи, където стойността на полето е равна на определена стойност.
- **Пример:** `{ "age": { "$eq": 25 } }`  
  Намери документи, където `age` е равно на 25.

### 2. `$ne`
- **Описание:** Съвпада с документи, където стойността на полето не е равна на определена стойност.
- **Пример:** `{ "age": { "$ne": 25 } }`  
  Намери документи, където `age` не е равно на 25.

### 3. `$gt`
- **Описание:** Съвпада с документи, където стойността на полето е по-голяма от определена стойност.
- **Пример:** `{ "age": { "$gt": 25 } }`  
  Намери документи, където `age` е по-голямо от 25.

### 4. `$gte`
- **Описание:** По-голямо или равно.
- **Пример:** `{ "age": { "$gte": 25 } }`

### 5. `$lt`
- **Описание:** По-малко от.
- **Пример:** `{ "age": { "$lt": 18 } }`

### 6. `$lte`
- **Описание:** По-малко или равно.
- **Пример:** `{ "age": { "$lte": 65 } }`

### 7. `$in`
- **Описание:** Съвпада, ако стойността е в зададен списък.
- **Пример:** `{ "status": { "$in": ["active", "pending"] } }`

### 8. `$nin`
- **Описание:** Стойността не е в зададения списък.
- **Пример:** `{ "status": { "$nin": ["deleted", "blocked"] } }`
